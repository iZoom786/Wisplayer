Phase 1: Master the Foundations

Foundational Skill Focus

Your time in Phase 1 should be heavily weighted towards SQL, the lingua franca of data. Python provides the logic, and the command line is your environment. This chart shows a recommended distribution of your learning focus.

SQL is Non-Negotiable

Focus on advanced `JOINs`, window functions, and subqueries. This is your most critical skill.

Python with Pandas

Learn core programming concepts and master the Pandas library for data manipulation.

Linux Command Line

Get comfortable in the terminal. You'll use it daily to interact with servers and services.
